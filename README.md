## ✨ 特性

- ✅ 使用 Vue.js 全家桶开发
- 🔴 网易云账号登录（扫码/手机/邮箱登录）
- 📺 支持 MV 播放
- 📃 支持歌词显示
- 🌎️ 可直接播放（需要登录网易云账号）
- 📻 支持私人 FM / 每日推荐歌曲
- ✔️ 每日自动签到（手机端和电脑端同时签到）
- 🟥 支持 Last.fm Scrobble
- 🌚 Light/Dark Mode 自动切换
- ☁️ 支持音乐云盘
- ⌨️ 自定义快捷键和全局快捷键
